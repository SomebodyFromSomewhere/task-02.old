"# task-02" 
Имеется поле N на M клеток. Определенные клетки заняты грядками. Соседствующие по
вертикали и горизонтали грядки образуют одну большую. Требуется вывести количество
грядок.
Пример #1:
Входной файл:
4 5
##...
.#.#.
.#.##
..#..
Выходной файл:
3
Пример #2:
Входной файл:
6 8
##...###
.#.#.#.#
.##.##..
..##..##
.......#
######..
Выходной файл:
5
